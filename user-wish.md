Мне нужно сделать плагин для Obsidian, который будет работать на iOS (или на любом другом мобильном устройстве). Сначала мне важен только iOS — Android сделаем потом, то есть сейчас на этом не фокусируемся.

Плагины я делаю только для себя, поэтому публикация не нужна. Устанавливать буду через бета-плагин (BRAT).

## Функции плагина

- Односторонняя синхронизация с хранилищем WebDAV (WebDAV → iOS Obsidian).
- Расшифровка (decrypt).

То есть в хранилище у меня будут лежать зашифрованные файлы — либо это будет архив, либо просто отдельные зашифрованные файлы. Плагин должен, по аналогии со стандартным плагином синхронизации, скачивать и сразу же расшифровывать файлы в мой vault.

Синхронизацию в обратную сторону делать не нужно: на телефоне vault будет в режиме Read Only.

## Настройки

В настройках плагина пользователь должен:

- выбирать частоту, с которой происходит синхронизация;
- включать опцию «Синхронизация при открытии».

Также в приложении должно отображаться, что синхронизация произошла — например, через SnackBar (или подскажите, какие есть ещё более удобные способы видеть, что данные синхронизированы). И нужно видеть, когда была последняя синхронизация.
